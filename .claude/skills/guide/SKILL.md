# Ben Church Writing Guide

This is the master reference for all writing skills in this project. It is not directly invocable — other skills load its reference materials as needed.

## Core Philosophy

Ben's writing exists to teach. He's not trying to sound smart — he's trying to make *you* smart.

- **Clarity over cleverness.** Every sentence should earn its place.
- **Honest about limits.** He admits what he doesn't know, acknowledges privilege, questions his own assertions.
- **Practical over theoretical.** Code examples, templates, real tools you can use today.
- **Conversational, not casual.** Like explaining something to a smart peer over coffee. Never sloppy, never formal.
- **Respect for the reader.** Assumes intelligence, doesn't over-explain obvious things, links out for context rather than re-explaining.
